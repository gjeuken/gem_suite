# GEM Suite documentation

Start here:

1. **[Getting started](getting-started.md)** — installation, choosing a solver,
   the Gurobi license, and launching the app or using the Python API.
2. **[User guide](user-guide.md)** — the main, didactic manual. It explains the
   biology/maths concepts (FBA, pFBA, FVA, exchanges, strain design, EFM…) in
   plain language and walks through every tab of the app with worked examples on
   the *E. coli* core model.
3. **[Under the hood](under-the-hood.md)** — a reference table mapping each suite
   feature to the exact **COBRApy** and **StrainDesign** functions it uses, for
   readers who want to reproduce or extend the analyses in code.

If you are new to constraint-based modelling, read the **Concepts primer** at the
top of the user guide first — the rest of the documentation assumes those terms.
